# DatabaseConnection
Данный проект содержит десктопное приложение, написанонное на Python, для взаимодействия с базой данных, которая может быть расположена как на локальном хосте, так и на удалённом.
В приложении реализован функционал просмотра, добавления, редактирования и удаления записей из БД.
